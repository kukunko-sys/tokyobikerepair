# スマホ特化 夜間自転車修理センターLP作成計画

本プロジェクトは、大田区を拠点とする夜間専門の自転車パンク出張修理サービスのスマホ向けLP（ランディングページ）を作成するものです。ユーザーの要望に沿って、保守性と堅牢性・AIO/SEO最適化を重視したバニラベース（外部ライブラリ非依存）のモダンな構成を使用します。

## 設計方針・アーキテクチャ
*   **AIO (AI Optimization) / SEO 最適化**: 
    *   `<header>`, `<main>`, `<section>`, `<footer>` 等のHTML5セマンティックタグを用いた厳格な文書構造。
    *   `h1` 〜 `h6` の正しい見出し階層の設定。
    *   `JSON-LD` (Schema.org) を用いた `LocalBusiness` および `Service` のメタデータの埋め込み（AIが事業内容や対応エリアを解析しやすくするため）。
*   **スマホファースト・モダンUI**:
    *   CSS変数（Custom Properties）を用いたカラーテーマ（安心感・信頼感のあるグリーン基調）の集中管理。
    *   フレックスボックス (`flex`) とグリッド (`grid`) を活用したフルードレイアウト。
    *   グラスモーフィズム効果や、ボタンのマイクロインタラクション（ホバー・タップ時のフィードバック）。
    *   フォントはGoogle Fontsの「Noto Sans JP」の複数ウェイト（例: 400, 700）を読み込み。
*   **技術スタック (依存関係のダイエット)**:
    *   標準APIファーストで、TailwindCSSなどのフレームワークは使用せずVanilla CSSで構築。
    *   JSが必要な場合もVanilla JSで記述（今回はほぼ表示と静的リンクのみのため不要または最小限）。

## 検証計画 (Verification Plan)

### 手動検証 (Manual Verification)
1. ローカルで `npx serve .` または `python -m http.server` 等を使用して `index.html` をブラウザで開く。
2. ブラウザの開発者ツール（F12）を開き、デバイスツールバー機能を使用してスマートフォンビュー（iPhone 14相当等）に切り替え、レイアウトが崩れていないか確認する。
3. すべてのリンク（特にLINE通話・チャットのリンク）がクリック可能で、適切にフィードバック（タップ効果）があることを確認する。
4. HTMLのValidatorツール（W3C Nu Html Checker等）またはリッチリザルトテストツールにコードを貼り付け、構造化データ・HTMLシンタックスにエラーがないことを確認する。
